# 墨喵书屋 — 前端全页面功能清单

> 统计日期：2026-06-27 | 技术栈：Vue 3 + Element Plus + Pinia + Vite

---

## 一、独立页面（无布局外壳）

### 1. 登录页 `/login`
- 邮箱 + 密码登录
- "记住我"自动登录（Token 持久化）
- 切换暗黑模式
- 跳转注册页
- 毛玻璃卡片居中布局

### 2. 注册页 `/register`
- 邮箱 + 邮箱验证码（60s 倒计时）
- 用户名 + 密码 + 确认密码
- 性别选择（男/女/未知）
- 生日选择
- 图形验证码（Canvas 绘制，点击刷新）
- 注册按钮 → 成功后跳转登录
- 系别和用户类型已隐藏（后端默认值）

### 3. 分享重定向 `/s/:isbn`
- 短链解析，重定向到书籍详情或首页

## 二、主布局内页面（BasicLayout 嵌套）

### 4. 首页 `/home`
- 热门书籍推荐（3D 书本翻转展示）
- 天气氛围面板（7 种天气可切换）
- AI 每日画报展示
- 动态壁纸引擎（模糊漫游 + 预设壁纸库）
- 鼠标拖尾光绘效果
- 新手引导（Driver.js）
- 塔罗牌书籍推荐
- 右下角 AI 宠物挂件（PetWidget）

### 5. 书架 `/bookshelf`
- 书架分组管理（在读/已读完/想读/自定义）
- 拖拽排序 + 右键菜单（借阅/收藏/分享/分组）
- 3D 翻书飞出动画 → 跳转阅读器
- AI 管家节流（同书 5 分钟内不重复调用）
- 书籍搜索（按书名/作者/ISBN）
- 借阅/归还操作
- 热门推荐切换

### 6. 阅读器 `/book/reader`
- **章节阅读**：分页展示、左右翻页、章节缓存预加载
- **背景主题**：多预设背景色切换
- **字体调节**：字号 + 字体选择
- **AI 菜单栏**（右上角 ⚡）：
  - AI 智能沉浸氛围（天气粒子 + 背景音效）
  - AI 章节智能总结
  - AI 人物关系分析 → 知识图谱可视化
  - 生成章节题目 / 开启自动出题
  - 识别屏幕内容
  - 添加书签 / 查看书签
  - 🎧 打开智能有声书 → 管理员专属（`v-if="currentUserType === 3"`）
- **段落交互**：点击段落高亮 → 段评面板（分页+发表）
- **右键菜单**：
  - 🔊 从选中位置朗读 / 📖 全章朗读（TTS 批量并行 + SSE 流式播放）
  - 📄 复制文本
  - 📤 分享选段到论坛
  - 📝 添加笔记
  - 🎨 AI 文生图
  - 📚 生成选段题目
  - 🔍 词典查询
- **朗读悬浮窗**：倍速调整、拖拽定位、进度显示
- **笔记批注**：选中文本 → 写批注 → 保存到笔记库
- **分享图片生成**：选中文本 → 生成精美分享卡片（html2canvas）
- **管理员功能**：编辑章节 / 添加章节（`v-if="currentUserType === 3"`）
- 阅读进度自动同步 + 阅读时长统计
- 共读舱静默联动（笔记自动推送到共读房间）
- 知识图谱可视化（星球节点 + 连线）
- 漂流瓶系统集成
- 阅读器内 TTS 朗读（全章/选中，批量 SSE 流式）

### 7. 书籍详情 `/book/detail`
- 书籍封面 + 信息（书名/作者/评分/简介/AI 摘要）
- 星级评分
- 借阅/收藏/取消收藏
- 书籍评论列表（瀑布流双栏布局）
- 发表评论弹窗（评分 1-5 星 + 文字）
- 评论删除（本人/管理员）
- 加载更多评论

### 8. 论坛 `/forum`
- **帖子流**：发帖/浏览/搜索
- **评论系统**：
  - 发表评论（支持 @艾特好友、图片上传最多 9 张）
  - 回复评论（嵌套子评论）
  - 点赞/取消点赞
  - 评论图片预览（≤2 张展开 / >2 张折叠遮罩）
- **书籍卡片系统**：`[{book:ISBN}]` 格式解析 → 5 种卡片渲染
  - 单卡片：直接展示
  - 多卡片：左右箭头水平滚动轮播
- **分享卡片**：选中文本 → 生成精美分享图片

### 9. 聊天 `/chat`
- WebSocket 实时私聊
- 文本/图片/语音/视频消息
- 消息撤回、引用回复
- 好友搜索/添加/删除
- 好友申请处理
- 会话列表 + 未读红点

### 10. 消息中心 `/message`
- SSE 实时通知推送
- 通知类型：好友申请、消息、评论回复
- 通知列表 + 已读/未读状态
- 点击跳转到对应页面

### 11. 个人中心 `/profile`
- 用户头像/昵称/ID
- 🏆 成就入口（Lv.X 徽章，点击跳转成就中心）
- 阅读时长统计展示
- 个人介绍（bio）
- 编辑资料入口
- 个人信息表单（头像/昵称/密码修改等）
- 借阅记录入口
- 我的笔记/论坛评论/书籍评论入口
- 导出功能入口

### 12. 用户信息编辑 `/userInfo`
- 修改个人信息（昵称/性别/生日/介绍）
- 修改密码
- 头像更换

### 13. 借阅记录 `/borrowInfo`
- 借阅历史列表
- 当前在借/已还/逾期状态展示

### 14. 用户对外主页 `/user/profile`
- 查看其他用户的公开信息
- 阅读记录、收藏、评论展示
- 添加好友入口

### 15. AI 阅读周报 `/weeklyReport`
- AI 生成的阅读周报
- 阅读数据分析（时长/本数/连续天数）
- 阅读习惯建议 + 书籍推荐

### 16. 阅读历史画廊 `/read-gallery`
- GitHub 风格阅读热力图
- 阅读历史时间线展示

### 17. 我的笔记 `/profile/notes`
- 读书笔记列表
- 按书籍/时间排序
- 笔记详情查看

### 18. 我的论坛评论 `/profile/forum-comments`
- 论坛评论历史列表
- 按时间排序

### 19. 我的书籍评论 `/profile/book-comments`
- 书籍评论历史列表
- 评分 + 内容展示

### 20. 设置 `/settings`
- 暗黑模式开关（View Transition API 圆形扩散动画）
- 毛玻璃 / 液态玻璃材质切换
- 壁纸引擎开关
- ☕ 书籍共读胶囊舱开关
- 天气氛围开关
- 鼠标拖尾特效开关
- 新手引导开关

### 21. 管理员后台 `/admin`（仅 `userType === 3`）
- **6 个 Tab**：
  - 📚 书评审核：列表 + 通过/拒绝
  - 📝 段评审核：列表 + 通过/拒绝
  - 💬 论坛评论审核：列表 + 通过/拒绝 + 🖼图片预览
  - 🍾 漂流瓶审核：列表 + 放行/销毁
  - 👤 用户管理：搜索 + 冻结/解冻按钮（`canUse` 状态切换）
  - 📖 书籍管理：搜索 + 删除
- 待审核数量顶部统计卡片

### 22. 成就中心 `/achievements`
- 等级进度条（Lv.X + EXP 进度）
- 徽章网格（BadgeItem 组件）
  - 毛玻璃卡片 + 稀有度发光边框
  - 已解锁：彩色 + 悬浮动效
  - 未解锁：灰色 + 🔒 锁定图标
- 点击徽章 → 详情弹窗（名称/描述/解锁日期）
- 全局解锁动效（AchievementUnlocker 组件 + canvas-confetti 烟花）

### 23. 有声书图书馆 `/audiobook`
- 有声书书架网格卡片
- 按书名搜索
- 封面 + 书名 + 作者 + 总时长 + 已完成章节数

### 24. 书广场 `/bookSquare`
- 书籍广场浏览
- 书籍详情 `/bookSquare/detail/:id`

## 三、全局组件

### BasicLayout.vue
- 左侧导航栏（菜单项根据 userType 动态显示）
- 顶部搜索栏
- AI SSE 全局中枢（connectSSE → PET_AI_START/CHUNK/DONE）
- 全局右键菜单
- 壁纸引擎
- 天气氛围
- 暗黑模式切换
- AI 发送中枢（`provide('sendAiMsg')`）
- 用户类型注入（`provide('currentUserType')`）
- 共读舱胶囊入口（右下角悬浮）

### PetWidget.vue
- 右下角 AI 宠物浮动挂件
- 气泡对话式交互（chat/thinking/interaction/static/talk 五种气泡类型）
- 流式 SSE 逐字渲染（5s 自动消失）
- 预设问候语轮换
- 确认气泡（交互式选择）
- 本地关键词匹配引导回复

### AudiobookPlayer.vue
- 有声书播放器（章节列表侧边栏）
- 播放/暂停/上一段/下一段
- 进度条拖拽
- 倍速控制
- 单段音频生成按钮 + 📚 批量生成全书按钮
- 角色标注显示（旁白/青年男/青年女等）
- 播放进度记忆

### CoReadingRoom.vue
- 书籍共读舱（右下角胶囊入口 → 侧边抽屉）
- 创建/加入共读房间
- SSE 实时成员列表（ROOM_META 推送）
- 在线人数 + 成员头像
- 聊天消息（历史消息回放）
- 书摘卡片自动推送
- 退出房间 + 空房间 10s 自动销毁
- 浏览器关闭时紧急上报退房

### AchievementUnlocker.vue
- 全局弹窗遮罩（z-index: 9999）
- bounce-in 动画 + canvas-confetti 烟花
- 稀有度分级特效（传说级炫彩呼吸灯）
- 队列机制：连续解锁多个徽章逐个弹出

### BadgeItem.vue
- 单个徽章卡片组件
- 稀有度发光体系（bronze/silver/gold/legendary CSS class）
- 悬浮放大 + 旋转微动效
- emoji 占位图标（🥉🥈🥇👑）

## 四、Pinia Store

| Store | 职责 |
|-------|------|
| `userStore` | 用户登录态、Token、userId、userType、暗黑模式、阅读状态 |
| `petStore` | AI 消息管理（5 种气泡类型）、流式追加、自动出题、问候语 |
| `readerStore` | 阅读器状态、TTS 朗读、缓存、漂流瓶 |
| `achievementStore` | 徽章数据、等级/EXP、解锁队列、弹窗控制 |
| `appSettingsStore` | 全局设置（壁纸/天气/材质/胶囊舱等开关） |

## 五、技术特性汇总

| 特性 | 实现 |
|------|------|
| 路由守卫 | `beforeEach` 检查登录态 + Token 恢复 + admin 权限拦截 |
| 响应式 | `@media (max-width: 768px)` 移动端折叠布局 |
| 暗黑模式 | View Transition API 圆形扩散动画 + CSS 变量 |
| 毛玻璃 UI | `backdrop-filter: blur()` + `rgba` 背景 |
| 实时通信 | WebSocket（聊天）+ SSE（通知/有声书/共读/AI气泡） |
| 流式渲染 | AI 回复逐字出现（SSE → petStore → PetWidget） |
| 文件上传 | 论坛图片（FormData 上传，最多 9 张） |
| 图片导出 | html2canvas（分享卡片） + canvas-confetti（成就烟花） |
| 3D 动效 | CSS 3D transform（书架翻书）、Canvas 粒子（天气/拖尾） |
| 持久化 | Pinia persist（userStore/achievementStore/appSettingsStore） + localStorage |
| 中文 TTS | edge-tts 批量并行 SSE 流式 + 前端分句播放 |
| AI 限流 | AiRateLimiter（每分钟 10 次 + 每日 20 次 + 图片每日 5 张） |
| 内容审核 | 评论/漂流瓶默认待审 → 管理员通过/拒绝 |
